## Example 1
食べさせたくなかった

--食べさせたくない [形容詞]

----食べさせたい [形容詞]

----食べさせる [たい]

------食べる [させる]

------食べる [させる]

------食ぶ [る]


食ぶ --[る]--> 食べる --[させる]--> 食べさせる --[たい]--> 食べさせたい --[形容詞]--> 食べさせたくない --[形容詞]--> 食べさせたくなかった


食べる --[させる]--> 食べさせる --[たい]--> 食べさせたい --[形容詞]--> 食べさせたくない --[形容詞]--> 食べさせたくなかった
